# Contributing

This project is published as independent research for open discussion and development.

## Welcome

- Questions about the theory or formal model
- Critique of assumptions, definitions, or proofs
- Proposed extensions (new adversary models, new application domains)
- Bug reports on the proof of concept
- New evaluation experiments
- Documentation improvements

## How to Contribute

1. Open an issue describing what you want to discuss or propose
2. For code contributions, fork the repo and submit a pull request
3. All code must include tests
4. All written content must follow the writing standards in CLAUDE.md

## Writing Standards

This project enforces specific writing standards to ensure clarity and avoid
AI-detection patterns. See CLAUDE.md for the full list. Key rules:
- No banned words (leverage, utilize, streamline, comprehensive, robust)
- Maximum one em dash per document
- Pin all dependencies with exact versions
